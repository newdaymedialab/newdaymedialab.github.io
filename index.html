Project

NewDay Media Lab is building a platform for creators and storytellers: upload a short clip, pick a character, and receive a new video reconstructed from that character's point of view. We focus on short, high-quality outputs first to optimize cost and user experience.

How it works

Character detection → scene reconstruction → POV synthesis → post-processing

First release

Prototype for 5–59s videos with quick turnaround for demos and beta users.

Roadmap

Phase 1 — MVP
GitHub Pages demo, basic upload UI, short-video outputs (0–3 months)
Phase 2 — Beta
Integrate AI pipeline, automated processing queue, closed beta (3–6 months)
Phase 3 — Fund & Scale
Community funding, token launch, cloud GPU scaling (6–12 months)
Phase 4 — Pro
Full commercial API, longer videos, B2B partnerships (12+ months)
